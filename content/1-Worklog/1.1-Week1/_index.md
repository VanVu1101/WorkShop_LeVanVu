I studied the basic concepts of Cloud Computing, including:

# Challenges Encountered

During the first week, I encountered several challenges:

- AWS provides a wide range of cloud services, making it difficult to understand everything in a short time.
- Many cloud computing concepts were new to me.
- Understanding the relationships among AWS services required additional study.

---

# Solutions

To overcome these challenges, I:

- Read AWS official documentation.
- Reviewed Bootcamp learning materials regularly.
- Organized personal notes by service category.
- Discussed questions with the mentor and teammates.
- Practiced summarizing key concepts after each study session.

---

# Learning Outcomes

After completing Week 1, I was able to:

- Explain the basic concepts of Cloud Computing.
- Describe the AWS Global Infrastructure.
- Identify AWS core service categories.
- Understand the internship workflow and Bootcamp roadmap.
- Build a strong theoretical foundation for future AWS hands-on laboratories.

The first week focused on orientation and building theoretical knowledge.
