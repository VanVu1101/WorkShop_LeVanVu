# Week 6 Objectives

The objectives for Week 6 were:

- Understand Amazon RDS architecture and database deployment workflow.
- Configure Amazon VPC, Subnets and Security Groups.
- Deploy and connect Amazon EC2 instances with Amazon RDS databases.
- Install and configure application environments on Amazon Linux.
- Deploy Node.js applications connected to cloud databases.
- Learn Amazon Lightsail service and simplified cloud deployment.
- Deploy WordPress, PrestaShop and Akaunting applications using AWS Lightsail.
- Configure application security and access control.
- Create snapshots and restore cloud resources.
- Monitor cloud resources and optimize infrastructure usage.
