## 5. AWS Cost Management and Cloud Architecture

Studied cloud cost management and infrastructure optimization.

Topics covered:

- AWS Budgets
- AWS Cost Explorer
- Billing Alerts
- AWS Pricing Calculator
- Cost and Usage Reports
- Resource Cleanup

Compared two deployment models:

### Traditional Architecture

- Amazon EC2
- Amazon RDS
- Amazon S3
- Amazon CloudFront

### Serverless Architecture

- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon CloudFront

Through cost estimation exercises, I learned that serverless architecture can significantly reduce infrastructure costs for small applications.
